Print Media

Print Media is a rule for how a web page will look when it's being printed, typically with Print Media you want to have easy to read styling that wont take up large amounts of ink and has unessentials things removed such as links or navigation. Print Media is done using Media Queries which we will cover later on, this topic will focus on what is essential and non-essential for Print Medias and best practices around it.
